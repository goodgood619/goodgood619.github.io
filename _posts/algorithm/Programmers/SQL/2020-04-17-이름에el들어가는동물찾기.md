**like**옵션을 줘서 문제를 풀수 있다. **%는 와일드카드 *과 동일하다**는 것을 기억하자  

## by SQL

```sql
-- 코드를 입력하세요
SELECT ANIMAL_ID, NAME
from animal_ins
where name like '%el%' and animal_type like 'Dog'
order by name
```

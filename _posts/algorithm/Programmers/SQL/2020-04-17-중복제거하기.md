**count**명령어는 중복제거를 해줄수는 없다. 그래서 **distinct** 명령어를 이용해서 체크를 해줘야 한다  

## by SQL

```sql
-- 코드를 입력하세요
SELECT count(distinct name)
from animal_ins;
```
